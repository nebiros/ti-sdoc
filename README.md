# Ti::Sdoc

Appcelerator's Titanium Mobile API parser, it dumps a SDOC file
that can be used in NetBeans for code completion or Aptana.

## Installation

Add this line to your application's Gemfile:

    gem 'ti-sdoc'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ti-sdoc
